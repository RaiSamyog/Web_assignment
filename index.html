WEB SCHOOL

Introduction

Samyog Rai- Owner of the website

Web school is a website that helps web developers and students help understand the importance of web development. This site focuses on helping developers in creating ideas for their website development. It'll have sources for that and some tutorials. There will be web development

terms included and comparision of two web development jobs. This website will have a review section for the users to review see the reviwes about it.
